음악 추천 사이트

과제 개요

본 프로젝트는 HTML, CSS, JavaScript를 활용하여 제작된 음악 추천 플랫폼입니다. 다양한 사용자 취향을 고려한 음악 필터링 및 인터랙티브 평점/즐겨찾기 기능을 실제 동작 가능하게 구현하였습니다.

핵심 기술 & 디자인 포인트

  • SPA(Single Page Application) 구조: 탭 이동 시 새로고침이 없는 매끄러운 화면 전환
  • Spotify 스타일 UI: 매끄러운 다크 모드와 네온 그린 액센트 컬러의 완성도 높은 테마
  • 데이터 영속성: 브라우저를 활용해 즐겨찾기 및 평점 데이터 저장 유지
  • 반응형 그리드: 모바일, 태블릿, PC 해상도를 완벽히 지원하는 유연한 CSS Grid
